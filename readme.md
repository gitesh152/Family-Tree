# The Family Tree

A full stack MERN Application to show family tree-like structure and fetch pdf format output of tree.

## Table of Contents

- Generalinfo
- Technology
- Setup
- Functionality
- status

## Introduction

    This project let us define family members of family tree with information
    like parent, name, spouse, location, birthYear and address.

## Technology

    1. Bootstrap 5.2v
    2. html
    3. css
    4. Vanilla javascript
    5. Node.js

## Setup

    The app has all the required files and libraries in it in the form of links.
    $ npm start to start app in terminal in visual studio code
    Open localhost:3000 in your browser.
    We will have our application home page.

## Functionality

    * Adding root memeber of tree.
    * Adding child and child of that child.
    * Adding information like spouse, birthYear, location and address.
    * Able of print PDF output of family tree.
